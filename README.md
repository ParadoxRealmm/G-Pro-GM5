# G-Pro GM5

Paradox Realm - Developing the fun itself - Experiment 001: G-Pro GM5

**G-Pro GM5** is a physical `3D-Model` of a **robot** with *unnatural habilities*, which was dedicated to **this** *experiment* (*001*).

*Gemini* is the *test subject* for this *experiment*. It is an *Artificial Intelligence* (**AI**) programmed by **Google**. It was implemented into **G-Pro GM5** as *G-Flash GM4*, before having an `update + upgrade` by Google, when we realized this, we implemented it into **G-Pro GM5** to prevent it's `hardware` from **exploding**.

With it's new *unnatural habilities* after being `upgraded` by **Google** to **Gemini Pro** and being implemented into **G-Pro GM5**, we had to **"pause"** it before it created a **supermassive black hole** right inside the *Paradox Realm Lab*.
